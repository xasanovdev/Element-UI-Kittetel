## Button Customization Options

### Types

- **button-primary:**

  - active
  - hover
  - disabled
  - Available colors:
    - white
    - blue
    - green
    - orange
    - red
    - gray

- **button-pain-primary:**
  - active
  - hover
  - disabled
  - Available colors:
    - white
    - blue
    - green
    - orange
    - red
    - gray

### Sizes

- Available sizes:
  - large
  - medium
  - small
  - mini

### Border Radius

- Available border radius options:
  - rounded-full
  - not-rounded

### Change Place Icon

- Option to reverse to the `.button-content` wrapper
  - reverse

### Border

- Available border options:
  - border-none
